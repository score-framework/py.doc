Virtual Environment
===================

The virtual environment does not need anything fancy, so you can just create it
inside the project folder:

.. code-block:: console
  
  $ mkdir score-test
  $ cd score-test
  $ python3 -m venv virtualenv
    ...
  $ source virtualenv/bin/activate

<tutorial_setup_module>`.
